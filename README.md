# 12주차 과제: 시계열 예측 기반 비트코인 트레이딩 전략

## 1. 과제 개요
본 과제는 시계열 데이터를 활용하여 비트코인 가격의 변화 방향을 예측하는
딥러닝 모델을 구축하고, 이를 기반으로 간단한 트레이딩 전략을 설계한 뒤
Buy and Hold 전략과 성과를 비교·분석하는 것을 목표로 한다.

## 2. 모델 설계 및 학습
실습 자료에서 제공된 LSTM 기반 딥러닝 모델을 활용하여
비트코인 가격 데이터로 학습을 진행하였다.
과거 일정 기간의 가격 정보를 입력으로 사용하여,
다음 날 가격이 상승할지 하락할지를 예측하는
이진 분류 문제로 모델을 구성하였다.

## 3. 투자 전략 설계
모델의 예측 결과를 활용하여 다음과 같은 단순한 트레이딩 전략을 적용하였다.

- 다음 날 가격이 상승할 것으로 예측될 경우 매수
- 하락이 예측될 경우 매수하지 않고 현금 보유

복잡한 매매 규칙보다는 모델 예측의 방향성 자체가
투자 성과에 어떤 영향을 미치는지 확인하는 데 초점을 두었다.

## 4. 벤치마크 전략
비교를 위한 벤치마크로는 Buy and Hold 전략을 사용하였다.
이는 분석 기간의 시작 시점에 비트코인을 매수한 후,
추가적인 거래 없이 보유를 유지하는 방식이다.

## 5. 실험 결과 및 분석
모델 기반 트레이딩 전략과 Buy and Hold 전략의 누적 수익률을 비교한 결과,
두 전략 간의 성과 차이를 확인할 수 있었다.

단순한 모델과 전략임에도 불구하고,
일부 구간에서는 가격 변동 방향을 반영하여
벤치마크와 다른 수익 흐름을 보이는 것을 확인하였다.
다만 급격한 변동성이 발생하는 구간에서는
예측 성능의 한계 또한 관찰되었다.

## 6. 결론 및 고찰
본 과제를 통해 시계열 예측 모델을 실제 트레이딩 전략에
적용해보는 과정을 경험할 수 있었다.
향후에는 추가적인 지표나 더 복잡한 모델을 활용하여
예측 성능 및 투자 전략을 개선할 수 있을 것으로 기대된다.
