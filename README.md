## 과제: https://github.com/juho127/TimeSeriesForecastingTest 를 참고하여(git fork) 자신의 repo에 과제노트북파일과 실습 내용을 기반으로 1. 모델을 설계하고 훈련한다음 2. 투자전략을 설계하여 적용하고 3. 벤치마크 전략과 비교하여 분석

## 1. 모델을 설계하고 훈련
실습 자료에서 제공된 LSTM 기반 딥러닝 모델을 활용하여 비트코인 가격 데이터로 학습을 진행했습니다.
과거 일정 기간의 가격 정보를 입력으로 사용하여, 다음 날 가격이 상승할지 하락할지를 예측하는 이진 분류 문제로 모델을 구성했습니다.

## 2. 투자 전략 설계
모델의 예측 결과를 활용하여 다음과 같은 투자 전략을 설계했습니다.

- 다음 날 가격이 상승할 것으로 예측될 경우 매수
- 하락이 예측될 경우 매수하지 않고 현금 보유

복잡한 매매 규칙보다는 모델 예측의 방향성 자체가 투자 성과에 어떤 영향을 미치는지 확인하는 데 초점을 두었습니다.

## 3. 벤치마크 전략과 비교 분석
비교를 위한 벤치마크로는 'Buy and Hold 전략: 분석 기간의 시작 시점에 비트코인을 매수한 후, 추가적인 거래 없이 보유를 유지하는 방식'을 사용했습니다.

해당 벤치마크 전략과 설계한 모델의 전략에 따른 누적 수익률을 비교한 결과,

1) 단순한 모델과 전략임에도 불구하고, 일부 구간에서는 가격 변동 방향을 반영하여 벤치마크와 다른 수익 흐름을 보이는 것을 확인.
2) 다만 급격한 변동성이 발생하는 구간에서는 예측 성능의 한계 존재.

위와 같은 분석 결과를 도출할 수 있었습니다.
